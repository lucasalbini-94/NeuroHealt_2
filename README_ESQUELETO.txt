NeuroHealth Desktop - Esqueleto para alumnos

Este proyecto conserva la estructura general de la aplicación, pero deja la funcionalidad principal pendiente de implementación.

Incluye:
- Dominio y enums.
- Interfaces.
- Servicios con firmas de métodos y comentarios TODO.
- Repositorios con firmas de métodos y comentarios TODO.
- Formularios Windows Forms simplificados y listos para completar.
- Program.cs y archivo .csproj.

Objetivo: que el proyecto compile como punto de partida y que los estudiantes completen la lógica solicitada en el enunciado.

los archivos .txt de prueba dentro del archivo neurohealth_txt_prueba.zip hay que ubicarlos en la siguiente carpeta bin\Debug\netX.X-windows\Datos
